## 需求总览

### 核心业务规则

- **包月制：** 住户按期为车辆缴费（月/季/年），系统记录各车辆各周期的缴费状态
- **一户多车 + 排序定档：** 每户登记的车辆有排序序号，不同序号对应不同收费标准
  - 例：第1辆 200元/月，第2辆 300元/月，第3辆 500元/月
- **缴费周期：** 按月、按季、按年 — 每辆车单独选择续费周期
- **收费策略可配置：** 每个档位的月费/季费/年费可独立设置，支持优惠策略（新能源折扣、特定房号免费等）
- **车辆三种交费状态（动态计算，不存储）：** **免费状态有精确起止日期（非仅月份）**

  | 状态 | 含义 | 判定规则（精确到日） |
  |------|------|---------|
  | 🟢 **免费** | 已缴费且在有效期内 | 今天 在 [paid_from, paid_to] 之间 |
  | 🟡 **临时** | ① 未登记的访客车辆<br>② 已登记但从未缴费<br>③ 已过期但在宽限期内 | 未登记 或 无缴费记录 或 今天 > paid_to 且 (今天 - paid_to) <= 宽限期天 |
  | 🔴 **过期** | 超过有效期且超过宽限期 | 今天 > paid_to 且 (今天 - paid_to) > 宽限期天数 |

  **关键语义：**
  - `paid_from` = 缴费周期起始月的第一天（如 period_start=2026-06 → 2026-06-01）
  - `paid_to` = 缴费周期截止月的最后一天（如 period_end=2026-06 → 2026-06-30）
  - 🟢 免费状态覆盖 [paid_from, paid_to] 整段日期，**不向前覆盖历史**
  - 临时车转包月：缴费前的临时停放天数 **另外计算**，不因缴费而"洗白"

  **宽限期可配置：** 系统设置项 `grace_period_days`，默认15天
  **临时停车阶梯费率：** 系统设置项 `temp_parking_rules`（JSON），默认30分钟免费、24小时封顶20元

- **三权分立权限管理：** 操作员、管理员、超级管理员三级角色
- **欠费管理：** 显示当前周期已缴费/欠费车辆清单

### 数据模型

```
用户 (User)
├── id
├── 用户名 (unique)
├── 密码哈希 (bcrypt)
├── 显示名称
├── 角色 (super_admin / admin / operator)
├── 是否启用
├── 创建时间, 最后登录时间

住户 (Resident)
├── id
├── 房号 (unique)
├── 业主姓名
├── 联系电话
├── 备注
├── 创建时间, 更新时间
└── 关联车辆 (按 sort_order 排序)

车辆 (Vehicle)                       ← **交费状态不存储，动态计算**
├── id
├── 车牌号 (unique)
├── 品牌 (可选)
├── 颜色 (可选)
├── 车辆类型 (小车/大车/摩托/新能源 — 用于优惠策略匹配)
├── 排序序号 (sort_order)          ← 关键：该户第几辆车
├── 注册状态 (正常/停用)
├── resident_id → Resident (nullable — null=未登记的临时车辆)
├── 创建时间, 更新时间
└── 缴费记录 → PaymentRecord[]

⤷ 交费状态计算（实时，不落库，精确到日）：
   IF resident_id IS NULL → "临时"
   ELSE IF 无缴费记录 → "临时"
   ELSE:
     paid_from = 1日(period_start)     # "2026-06" → 2026-06-01
     paid_to = 末日(period_end)         # "2026-06" → 2026-06-30
     IF today <= paid_to → "免费"       # 在有效期内
       status_start = paid_from, status_end = paid_to
     ELIF (today - paid_to) <= grace_days → "临时"  # 宽限期内
       status_start = paid_to + 1d, status_end = paid_to + grace_days
     ELSE → "过期"
       status_start = paid_to + grace_days + 1d, status_end = null

缴费记录 (PaymentRecord)           ← 标记某车某周期已缴费
├── id
├── vehicle_id → Vehicle
├── 周期起始 (如 2026-05)
├── 周期截止 (如 2026-05)
├── 周期类型 (月/季/年)
├── 缴费金额
├── 适用规则摘要                   ← 记录当时用了什么档位+优惠
├── 缴费方式 (现金/微信/支付宝/转账)
├── 操作员 → User.id
├── 备注
├── 缴费时间

收费档位规则 (FeeTier)              ← 按序号定费率，**版本化，每次修改新增记录**
├── id
├── 序号档次 (seq_from, seq_to)    ← 如 1~1, 2~2, 3~999
├── 月费 (元)                       ← 月缴单价
├── 季费 (元)                       ← 季缴单价（可设置，不填则 = 月费 × 3）
├── 年费 (元)                       ← 年缴单价（可设置，不填则 = 月费 × 12）
├── **生效日期 (effective_date)**    ← 从哪天开始执行，不可早于操作时间
├── **过期时间 (expired_at)**        ← 被新版本替代时自动设为操作时间，null=当前有效
├── 创建时间

⤷ 版本化规则：编辑 ≠ 修改原行，而是将旧行 expired_at = now，新建一行
  缴费时查询：取 effective_date <= 缴费周期 AND (expired_at IS NULL OR expired_at > 周期)

特殊优惠策略 (DiscountPolicy)       ← **同样版本化**，与 FeeTier 一致的版本管理
├── id
├── 优惠名称                        ← 如"新能源车优惠"
├── 优惠类型                        ← discount_percent / free / fixed_fee
│                                     discount_percent: 按百分比打折
│                                   free: 免费
│                                   fixed_fee: 固定金额
├── 优惠值                          ← 如 0.8（打八折），0（免费），100（固定100元）
├── 适用范围 (scope_type)           ← vehicle_type / room_prefix / all
├── 范围值                          ← 如"新能源"，"物业-" 等
├── 适用范围序号 (apply_to_seq_from, apply_to_seq_to)  ← 仅在某个序号范围内生效
├── **生效日期 (effective_date)**    ← 从哪天开始执行，不可早于操作时间
├── **过期时间 (expired_at)**        ← 被新版本替代时自动设置
├── 创建时间

操作日志 (OperationLog)             ← 记录关键操作，供超级管理员审计
├── id
├── 操作人 → User.id
├── 操作类型 (login/logout/create/update/delete/config等)
├── 操作对象 (如"住户 1-201"、"FeeTier 档位2")
├── 详情
├── IP地址
├── 操作时间

系统设置 (SystemSetting)             ← 键值对，全局可配置
├── id
├── 键 (key, unique)                ← 如"grace_period_days"、"company_name"
├── 值 (value)                      ← 如"15"、"之江花园"
├── 说明 (description)              ← "车辆过期宽限期(天)"
├── 更新时间
```

```

  选择车辆 → 查看当前状态（免费/临时/过期）

  → 系统自动计算金额(档位+优惠)

```

def require_role(*roles: str):
    """依赖注入：检查当前用户角色"""
    async def checker(request: Request):
        user = request.session.get("user")
        if not user:
            raise HTTPException(status_code=303, headers={"Location": "/login"})
        if user["role"] not in roles:
            raise HTTPException(status_code=403, detail="权限不足")
        return user
    return checker

# 使用示例

### Task 4: 收费策略配置（档位规则 + 优惠策略）— 版本化管理

**Objective:** 灵活配置收费策略（管理员/超级管理员），每次修改生成新版本，保留历史

**核心规则：**
- **版本化：** 编辑 ≠ 修改原行。旧行 `expired_at = now`，新建一行
- **生效日期限制：** `effective_date` 不可早于操作时间（防止回溯乱改费率）
- **缴费时费率查询逻辑：**

def vehicle_status(plate_number: str):
    vehicle = get_vehicle_by_plate(plate_number)
    grace_days = get_setting("grace_period_days", 15)
    temp_rules = json.loads(get_setting("temp_parking_rules", "{}"))

    status_info = calculate_payment_status(vehicle, grace_days, date.today())

    # 计算临时历史：注册日期到首次缴费之间的临时天数
    temp_history = None
    if vehicle and vehicle.resident_id:
        first_payment = get_first_payment(vehicle.id)
        if first_payment:
            paid_from = first_day(first_payment.period_start)
            if vehicle.created_at.date() < paid_from:
                temp_days = (paid_from - vehicle.created_at.date()).days
                # 按阶梯规则估算临时费用
                est_fee = calc_temp_parking_fee(temp_rules, temp_days * 24)
                temp_history = {
                    "from_date": vehicle.created_at.date().isoformat(),
                    "to_date": (paid_from - timedelta(days=1)).isoformat(),
                    "temp_hours": temp_days * 24,
                    "estimated_daily_charge": temp_rules.get("daily_cap", 0),
                    "estimated_fee": est_fee,
                    "note": f"按{temp_rules.get('daily_cap', 0)}元/天封顶计算，{temp_days}天共约{est_fee}元"
                }

    return {
        "plate_number": plate_number,
        "query_time": datetime.now().isoformat(),
        "is_registered": vehicle is not None and vehicle.resident_id is not None,
        "resident_info": resident_info if vehicle and vehicle.resident else None,
        "payment_status": status_info["status"],
        "status_start": status_info["status_start"],
        "status_end": status_info["status_end"],
        "status_detail": status_info["detail"],
        "temp_pricing": format_temp_pricing_for_api(temp_rules),
        "temp_history": temp_history,
    }

def format_temp_pricing_for_api(rules: dict) -> dict:
    """将阶梯规则格式化为 API 友好结构"""
    if not rules:
        rules = {"free_minutes": 0, "daily_cap": 0, "tiers": [], "overflow": "daily_reset"}

    # 生成人类可读描述
    parts = []
    free_min = rules.get("free_minutes", 0)
    daily_cap = rules.get("daily_cap", 0)
    if free_min > 0:
        parts.append(f"{free_min}分钟内免费")
    if daily_cap > 0:
        parts.append(f"24小时封顶{daily_cap}元")

    return {
        "free_minutes": free_min,
        "daily_cap": daily_cap,
        "tiers": rules.get("tiers", []),
        "overflow": rules.get("overflow", "daily_reset"),
        "description": "，".join(parts) if parts else "按阶梯计费"
    }

**费用计算函数：**
```python
def calc_temp_parking_fee(rules, total_hours):
    """
    根据阶梯规则计算临时停车费用
    rules: JSON 规则对象
    total_hours: 总停放小时数
    """
    free_min = rules.get("free_minutes", 0)
    daily_cap = rules.get("daily_cap", 0)
    overflow = rules.get("overflow", "daily_reset")
    tiers = rules.get("tiers", [])

    if total_hours <= free_min / 60:
        return 0  # 还在免费时段内

    if overflow == "daily_reset":
        # 按天分段计算，每天封顶
        days = math.ceil(total_hours / 24) if daily_cap > 0 else 1
        if daily_cap > 0:
            return days * daily_cap  # 简化：直接按天×封顶
        else:
            # 按天内阶梯计算
            daily_hours = min(total_hours, 24)
            return calc_tiered_fee(tiers, daily_hours) * days
    else:
        # 累计计算，不按天拆分
        return calc_tiered_fee(tiers, total_hours)

def calc_tiered_fee(tiers, hours):
    """按阶梯档位计算费用"""
    total = 0
    remaining = hours
    for tier in sorted(tiers, key=lambda t: t["from_hours"]):
        tier_span = tier["to_hours"] - tier["from_hours"]
        if remaining <= 0:
            break
        portion = min(remaining, tier_span)
        if tier.get("per") == "hour":
            total += portion * tier["fee"]
        else:
            total += tier["fee"]
        remaining -= portion
    return total
```

## Risks & Tradeoffs

| 风险 | 缓解 |
|------|------|
| 用户不理解 Web 应用模式 | 打包为 exe 后双击自动打开浏览器（`webbrowser.open`） |
| SQLite 数据丢失 | 程序启动时自动备份到 `backups/`（超级管理员可手动备份） |
| 月份周期跨年计算 | 统一用 `YYYY-MM` 字符串比较，按月升序排列 |
| 住户换房/换车牌 | 支持编辑车辆信息、编辑房号 |
| session 不安全 | 单机场景，local-only 访问，session 用 itsdangerous 签名 |
| 操作员误操作 | 所有缴费记录操作人可追溯，超级管理员可查日志 |
| 忘记密码 | 超级管理员可通过数据库直接重置（或留后门账号说明） |
