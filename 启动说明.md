# 停车费管理系统 - 启动说明

## ⚡ 快速启动（推荐）

### 🎯 最终用户：直接双击 exe 或运行安装包
- **便携版**：双击 `dist\parkman.exe`
- **安装版**：运行安装包，自动创建桌面快捷方式

### 🛠 开发者：源码运行

#### 首次搭建环境（只需一次）
双击 `dev_setup.bat`，自动创建虚拟环境并安装依赖。

#### 启动应用
双击 `start.bat`

浏览器访问 http://127.0.0.1:8080

## ❓ 常见问题

### 问题：提示 "ModuleNotFoundError: No module named 'uvicorn'"
**原因**：依赖未安装
**解决**：双击 `dev_setup.bat` 重建虚拟环境

### 问题：端口被占用
**解决**：编辑 `run.py`，修改端口号后重新启动

### 问题：启动后浏览器打不开页面
**检查**：
1. 确认终端显示 `Application startup complete`
2. 手动在浏览器输入 http://127.0.0.1:8080
3. 检查防火墙设置

### 问题：重装后数据还在吗？
**是的，数据不会丢失。** 数据库存储在用户数据目录（`%APPDATA%\停车费管理系统\`），卸载程序不会删除该目录。

### 问题：如何迁移到新数据库？
超级管理员登录后 → ⚙️ 系统设置 → **数据库管理** → 输入新的数据库文件路径 → 保存 → 重启应用。
