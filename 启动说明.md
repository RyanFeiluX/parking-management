# 停车费管理系统 - 启动说明

## ⚡ 快速启动（推荐）

### 🎯 最简单：直接双击
1. 打开文件夹 `d:\src\parking-management`
2. **双击 `run_simple.bat`** 文件
3. 等待应用启动
4. 浏览器访问 http://127.0.0.1:8080

| 方式 | 文件 | 说明 | 推荐度 |

## 🔧 命令行启动

### 步骤 1：安装依赖（只需一次）
```cmd
pip install fastapi uvicorn sqlalchemy jinja2 python-multipart itsdangerous openpyxl
```

### 步骤 2：启动应用
```cmd
python run.py
```

## ❓ 常见问题

### 问题：提示 "ModuleNotFoundError: No module named 'uvicorn'"

**原因**：Python 依赖包未安装

**解决方法**：
1. ✅ **最简单**：双击 `run_simple.bat` 使用系统 Python
2. 或运行命令：`pip install fastapi uvicorn sqlalchemy jinja2 python-multipart itsdangerous openpyxl`

### 问题：端口被占用

**解决方法**：
1. 编辑 `run.py` 文件
2. 修改端口号（如改为 8081）
3. 保存后重新启动

### 问题：权限不足

**解决方法**：
- 右键点击启动脚本，选择"以管理员身份运行"

### 问题：启动后浏览器打不开页面

**检查步骤**：
1. 确认终端显示 `Application startup complete`
2. 手动在浏览器输入 http://127.0.0.1:8080
3. 检查防火墙设置
